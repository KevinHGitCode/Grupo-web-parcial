Desarrollo Web Front-End 2025

Aprende las últimas tecnologías y herramientas para crear sitios web modernos y atractivos.

Ver programa

Bienvenidos al Curso

El curso de Desarrollo Web Front-End 2025 de Uniguajira está diseñado para formar profesionales con habilidades técnicas y creativas en el diseño y desarrollo de interfaces web modernas.

A lo largo del curso, aprenderás HTML5, CSS3, JavaScript y frameworks como Bootstrap para crear sitios web responsivos y atractivos. Además, conocerás las mejores prácticas de accesibilidad y rendimiento para garantizar experiencias de usuario óptimas.

Programa del Curso

Módulo 1: Fundamentos de HTML5

  • • Estructura básica de HTML5
  • • Etiquetas semánticas
  • • Formularios y validación
  • • Multimedia en HTML5

Módulo 2: CSS3 y Diseño Responsivo

  • • Selectores y propiedades CSS
  • • Flexbox y Grid
  • • Media queries
  • • Animaciones y transiciones

Módulo 3: Bootstrap y Frameworks

  • • Sistema de grid
  • • Componentes Bootstrap
  • • Personalización con Sass
  • • Optimización para producción
